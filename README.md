# Simulation of Gym

An OpenGL and GLUT-based graphics project that simulates a gym environment as part of earlier computer graphics coursework.

## Why this repository exists

I keep this repository public as an archive of hands-on graphics programming work. It represents an earlier phase of my engineering journey where I was exploring scene construction, rendering, and interactive graphics using lower-level tooling.

## What it demonstrates

- practical use of OpenGL with GLUT
- scene-oriented graphics programming
- coursework-level experimentation with interactive visual output

## Public positioning

This is a historical graphics project rather than a current flagship repository. I keep it visible to preserve the learning trail, but it should be interpreted as an academic artifact rather than a statement of my current production style.

## Ownership

I implemented this project as part of a graphics-focused learning exercise and retained it as part of my long-term technical archive.
